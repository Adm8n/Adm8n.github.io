dcpromo 输入管理员密码

两台 win7 作为被控主机​

基础配置：​

将两台win7 加入域控当中，我们将一台vmware 虚拟机进行克隆操作， 克隆完成后，找到C://Windows/System32/sysprep/sysprep.exe 双击运行windows7 重新封装， 同时更新每一台win7主机的SID（全球唯一）。 等待重启，之后进入用户登录界面， 先设置windows的管理员用户，和计算机名称这里更改为PC1 账户密码： Pa123456789 下一步 --> 选择工作网络。 如果选择错了网络类型：

     1. 打开控制面板 。
              2. 选择网络和共享中心 。
              3. 选择我们现在连接的网络。
                       4. 然后就可以在图示位置更改网络类型了。

计算机 --> 管理 --> 本地用户和组 删除多余的账户 PC1网络适配器配置

开始加入域控​

打开计算机的属性 --> 高级系统设置 -->计算机名/域更改 输入域控地址，然后输入域控服务器的账号密码 login: Administrator password: Admin123. 重启
